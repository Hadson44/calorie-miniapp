# calorie-miniapp

Telegram WebApp для трекінгу калорій.

## Структура
- `index.html` — статичний вхід у додаток і точка деплою.
- `app.js` — bootstrap-loader для основного коду з `src/scripts/app.js`.
- `style.css` — тонкий proxy-стиль, який імпортує `src/styles/style.css`.
- `src/scripts/app.js` — основна логіка застосунку.
- `src/styles/style.css` — основні стилі застосунку.
- `data/foods.js` — винесений каталог базових продуктів.
- `assets/` — локальні картинки, іконки та шрифти.
- `data/` — статичні дані й конфіги, які зручно виносити з коду.
- `docs/` — технічні нотатки по структурі проєкту.

## Нотатки
- Поточний стан користувача зберігається в `localStorage`.
- Структура залишає корінь сумісним зі статичним хостингом, але переносить основний код у `src/`.
